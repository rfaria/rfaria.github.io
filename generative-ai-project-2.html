Chatbot with Flask and RAG Integration

This project involved developing a chatbot using Flask, integrated with OpenAI and Retrieval-Augmented Generation (RAG) for ILUMEO. The objective was to enhance user interactions and the accuracy of responses by leveraging advanced AI technologies.

I took the initiative to build the chatbot, implementing OpenAI and RAG to vectorize data and improve query retrieval. By integrating these technologies, I developed a solution that could handle real-time user queries more efficiently. The model significantly enhanced the accuracy of responses, reducing query processing time and improving user satisfaction. The quick deployment demonstrated our ability to rapidly address user experience challenges and set a new standard for similar projects.
